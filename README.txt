Awesome Plugin to create professional, beautiful timeline and announcements on your site. 

== Description ==

Add a beautiful timeline section to your WordPress site quickly and easily.
Very configurable. Easy to setup. Great Support.

= Features =
	1. Responsive Timeline template (vertically aligned).
	2. Works on nearly all Themes.
	3. Customize the look and feel
	4. Translation Ready, i18n Support.
	5. Add announcement/post to display in Timeline.
	6. Add Image, Icon and date for announcement or posts.
	7. Unlimited icons to choose from.
	8. Show future, past or all announcements or posts.
	9. Display in ascending or descending order.
	10. Cool animation while scrolling of Timeline. 
	11. All content have configurable color and font settings.
	12. Show/Hide "Read More" link or button.
	13. Extensible via hooks
	
== Installation ==
= IMPORTANT = 

Please backup your WordPress database before you install/uninstall/activate/deactivate/upgrade Awesome WordPress Timeline Plugin.

1. 	Upload the folder awesome-timeline with its contents to wp-content/plugins/

2.  Activate the plugin.

3.	Upon activation, you will notice "Awesome Timeline" on Left Hand Side panel. 

4. Also, plugin will automatically create a post which will have a shortcode = [awesome-timeline] to display the timeline. But right now it wont display anything unless you do below steps.

5.  Click on the "Awesome Timeline" link on Left Hand Side and go to "Add Announcement" menu to add your content. After you have added announcement, you can view them by clicking "All Announcement".

6.  Click on "Awesome Timeline" and go to "Timeline" menu and enter title of that timeline. 

7.  Configure all properties for a Timeline and publish it. 

8.  Once you publish it, you can grab the shortcode and place it on a new Wordpress page/post or you can also view existing post created by plugin on step 4.

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'awesome-timeline'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Using FTP =

1. Download `awesome-timeline.zip`
2. Extract the `awesome-timeline` directory to your computer
3. Upload the `awesome-timeline` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
